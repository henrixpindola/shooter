Sobre o Projeto

Shooter Geometric é um jogo de nave 2D desenvolvido com HTML5 Canvas e JavaScript, inspirado no livro Desenvolva jogos com HTML5 Canvas e JavaScript.
O objetivo é proporcionar uma experiência divertida e didática, explorando conceitos de animação, colisão, física simples e interação com o usuário.

Tecnologias Utilizadas

  • HTML5, CSS3, JavaScript (ES6+)
  • Vite (build e dev server)
  • Node.js + Express.js (backend)
